每一个 C++ 表达式都有两个独立的属性：类型（type）和值类别（value category）。

🔗 值类别

C++ 有五（三）个值类别，其中两个是另外三个基本值类别的组合。

  • 基本类别
    • lvalue（左值）
    • prvalue（纯右值）
    • xvalue（亡值）
  • 混合类别
    • glvalue（泛左值）
      • 即 lvalue 或 xvalue。
    • rvalue（右值）
      • 即 prvalue 或 xvalue。

值类别具有以下性质：

  • 混合类别
    • glvalue（泛左值）
      • glvalue 可以通过左值到右值、数组到指针或函数到指针被隐式转换而转换成右值。
      • glvalue 可以是多态的：具有 glvalue 的表达式标识的对象的动态类型可以不与该表达式的静态类型相同。
      • glvalue 可以具有不完整类型。
    • rvalue（右值）
      • rvalue 不能被取地址。
      • rvalue 不能作为赋值或复合赋值运算符的左操作数。
      • rvalue 可以被用于初始化一个 const lvalue 引用。该 rvalue 表达式所标识对象的生存期被延长直到该引用的作用域结束。
